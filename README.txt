Importar el archivo busquedaautosuggest.sql a nuestra base de datos.
Para hacer que la aplicacion funcione, simplemente deberemos cambiar los datos del archivo dbcontroller.php por los datos de nuestra base de datos

Arrastrar todos los archivos y carpetas a la raiz de nuestro servidor (o carpeta donde queramos alojarlo) y entrar en la ruta
